- No bloco de notas:
```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello world :)");
    }
}
```

- Salve como HelloWorld.java, depois abra um terminal, navegue até o diretório do arquivo e:
```java
javac HelloWorld.java
java HelloWorld
```
